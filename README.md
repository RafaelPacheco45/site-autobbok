# AutoBook Landing Page

Landing page estática do AutoBook, comunidade social para lojistas de celulares do ecossistema AutoZap.

## Arquivos principais

- `index.html` - estrutura da página.
- `style.css` - estilos, responsividade e animações.
- `script.js` - menu mobile, scroll reveal, FAQ e interações.
- `assets/` - imagens otimizadas usadas no hero.

## Como abrir localmente

Basta abrir o arquivo `index.html` no navegador.

Também é possível servir a pasta com qualquer servidor estático, por exemplo:

```bash
npx serve .
```

## Publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste diretório para o repositório.
3. No GitHub, acesse `Settings > Pages`.
4. Em `Build and deployment`, escolha:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salve e aguarde o GitHub gerar a URL.
